## Framework/library used
**Frontend:**
- React.js
- UI: Tailwind CSS
  
**Backend:**
- Express.js
- Node.js
  
**Database:**
- MongoDB
  
**API:**
- Google Books API

**Others :**
- JSON Web Token
- Passport.js
- Axios

## Installation
```
git clone https://github.com/yuweiwang11/mehread.git
cd mehread
npm install

cd client
// at mehread/client
npm run dev

cd ../api
// at mehread/api
nodemon index.js
```
Go to http://localhost:5173
